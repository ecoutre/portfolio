The experience of flight booking is stuck in the 20th century. Across the board, the experience for a user to book a flight, especially on mobile, is terrible to say the least. The Hawaiian Airlines mobile application is no exception. As a redesign exercise I aimed to improve the user experience of the Hawaiian Airlines flight booking page to eliminate some of the anxiety and time spent searching by creating a simple, responsive, and elegant interface guiding the user on the way to purchase.
There were 5 main things I focused on redesigning the Hawaiian Airlines app:
1. Elegance
2. Guides/Navigates the user to completion
3. Takes advantage of the power of mobile gestures
4. Delightful to use & responsive
5. Consistent with Hawaiian Airlines brand

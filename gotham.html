For this project we were asked to build essentially an elementary version of Sim City. I was selected by my professors to be the team leader of a 5-person group to complete the task using Github to collaborate. The professors provided some code to get started, however my team decided to build everything on our own from scratch. The backend was written using multi-threaded agent code in Java. We had to learn most of the front-end on our own & wrote it using the Java Swing library. Unit tested using JUnit.
Project Info:
Team Size: 5
Duration of project: 5 weeks
Time Spent on Project: 50-60 hours per week (We didn't sleep much :) )
Lines of code: over 120,000
Language: Java
Tools: Eclipse, JUnit, Github, Git
Results: One of the highest group grades in the class
